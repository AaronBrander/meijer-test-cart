## Meijer Interview React project

A quick and simple build of a shopping cart. 
* Display products
* Must be logged in to add products to cart
* Sign in / Sign up pages, though their input doesn't do anything. Just click on the button.
* Add to cart
* View Cart
  * Remove from cart
